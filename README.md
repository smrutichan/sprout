# Sprout

AI-powered sustainability companion that helps users build eco-friendly habits through gamification.

## Features

- AI Climate Companion
- Carbon Impact Tracking
- Daily Climate Goals
- Streak System
- Achievements
- Eco Diary

## Tech Stack

Frontend:
- Next.js
- TypeScript

Backend:
- FastAPI
- SQLite
- SQLAlchemy
